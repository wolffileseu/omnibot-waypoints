The DMS (Development Menu System)

This Menu was originally designed as a tool to aid with waypointing plus goal creation without the need to remember all the commands & save manually typing.
This was further developed with more omnibot related commands
Nearly all the commands can be used as binds & it is recommended that you still use binds for the basic commands.
The DMS comes in here where there are more Omnibot commands than keys you can bind or remember
Although most of Omnibot commands are in the DMS this does not replace every command as some need additional parameters that need to be entered manually

It is highly recommeded that you do read the Wiki so you have an understanding of what the DMS does
https://mygamingtalk.com/wiki/OMNIBOT_-_Enemy_Territory_Modification
